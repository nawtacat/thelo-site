We're Building Our Team

Thelo is assembling a dedicated production team to create a world-class SAT Math course. We're looking for passionate, detail-oriented individuals to help us make learning more intuitive, interactive, and human.

Curriculum Lead (2 Positions)

Purpose

Design the course content map, define learning objectives, guide lesson sequencing, and ensure conceptual clarity.

Core Responsibilities

  • Break SAT Math into approximately 20 topic modules.
  • Define the structure for each lesson (objectives, pacing, key ideas).
  • Review scripts for pedagogical accuracy and clarity.
  • Ensure a scaffolded progression from easy to hard concepts.

Required Skills

  • Strong SAT Math knowledge
  • Pedagogical thinking (understands student pitfalls)
  • Ability to explain concepts step-by-step
  • Collaborative mindset

Key Tools

  • Google Docs / Notion
  • Optional: Desmos, GeoGebra for sketching

Selection Test Idea

"Design a 3-lesson flow for solving systems of equations, including key concepts and common student pitfalls."

Scriptwriter (2-3 Positions)

Purpose

Write the actual lessons—narration, MCQs, hints, and feedback—in a tone that is friendly, clear, and encouraging.

Core Responsibilities

  • Draft narration for each stage (intro, guided examples, wrap-up).
  • Create effective multiple-choice questions with clever distractors.
  • Write both positive and corrective feedback for student answers.
  • Iterate quickly using AI tools and feedback from the curriculum lead.

Required Skills

  • Clear, student-friendly writing style
  • Basic SAT Math knowledge
  • Willingness to co-write with AI and revise fast

Key Tools

  • Google Docs / Notion
  • ChatGPT or other AI writing assistants

Selection Test Idea

"Write an opening explanation and one multiple-choice question for a lesson on ratio problems. Use a friendly, casual tone."

Voice Talent (4 Positions)

Purpose

Bring Thelo lessons to life with diverse, warm, and clear voices across various accents and styles.

Core Responsibilities

  • Record clean, well-paced audio for lesson narration from provided scripts.
  • Follow strict file naming and delivery conventions.
  • Deliver recordings on time and be open to feedback and retakes.

Required Skills

  • Clear pronunciation and good microphone quality
  • Ability to read educational scripts in a natural, engaging way
  • A distinctive but listener-friendly accent

Key Tools

  • Microphone (USB or better) and a quiet recording space
  • Audio recording software (e.g., Audacity, GarageBand)

Selection Test Idea

"Record this 20-second script from a sample lesson in your natural tone and accent."

Drawing Artist + Integrator (1 Position)

Purpose

Translate narration into fluid handwriting animations and assemble the final lesson files for Thelo’s engine.

Core Responsibilities

  • Draw clear math explanations on a tablet to precisely match voiceover timing.
  • Organize and record pen stroke data for each distinct lesson stage.
  • Build and edit lesson JSON files, linking stages, audio, drawings, and feedback.
  • Perform basic functional tests on each lesson build to ensure it runs correctly.

Required Skills

  • Clean and quick math handwriting
  • Strong sense of timing and visual pacing
  • Basic JSON editing and troubleshooting

Key Tools

  • Drawing tablet and stylus
  • Thelo’s authoring/testing tools
  • Code editor (VS Code or similar)

Selection Test Idea

"Given a 30-second script, record pen strokes that match the voice, then submit a mock lesson JSON."

QA / Tester (1 Position)

Purpose

Act as the ultimate user advocate, ensuring every lesson is smooth, intuitive, and bug-free before it reaches students.

Core Responsibilities

  • Systematically test lesson playback on multiple devices (desktop and tablet).
  • Identify and report grading bugs, audio glitches, and visual inconsistencies.
  • Verify that handwriting recognition correctly grades answers.
  • Maintain a clear, detailed log of tested features, reported bugs, and their status.

Required Skills

  • Highly detail-oriented and methodical
  • Basic math fluency to spot incorrect logic
  • Clear and concise written communication

Key Tools

  • Google Sheets or Notion for logging
  • Access to test builds on Thelo
  • Desktop browser and a tablet

Selection Test Idea

"Test this sample lesson and report at least 3 distinct issues you would fix, providing timestamps or screenshots."

Project Manager (1 Position)

Purpose

The operational hub of the team, responsible for owning the timeline, unblocking creators, and ensuring a smooth production flow.

Core Responsibilities

  • Run efficient weekly check-ins to align the team and track progress.
  • Maintain a master tracker of all lessons, including owner, status, and due date.
  • Coordinate the handoff of assets between writers, voice artists, and integrators.
  • Proactively identify and communicate any blockers or risks to the timeline.

Required Skills

  • Exceptional, clear communication
  • Highly organized and proactive
  • Understands creative team dynamics
  • Comfortable in an async work environment

Key Tools

  • Discord or Slack for communication
  • Notion or Google Sheets for tracking
  • Google Drive for asset management
